Software Requirements

- PostgreSQL
- pgAdmin
- Power BI Desktop
- Microsoft Word
- Git

====================================================

Project Setup

1. Install PostgreSQL.
2. Restore or create the project database.
3. Execute SQL scripts in the sql folder.
4. Verify tables and data.
5. Open the Power BI dashboard.
6. Review documentation in docs and final_docs.

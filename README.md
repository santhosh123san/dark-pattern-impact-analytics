# 🛡️ Dark Pattern Impact Analytics & Trust Score System

> **An end-to-end Data Analytics project that evaluates application trustworthiness using SQL analytics, PostgreSQL, and Power BI dashboards.**

---

# 📌 Project Overview

Dark patterns are deceptive UI/UX techniques used by digital applications to manipulate user decisions. This project analyzes application reviews, calculates trust scores, classifies application risk levels, and generates business intelligence insights to help evaluate user trust and transparency.

The system combines **SQL analytics, PostgreSQL database management, business intelligence reporting, and Power BI dashboards** to transform raw data into actionable insights.

---

# 🎯 Project Objectives

* Analyze application review data
* Calculate Trust Scores for applications
* Classify applications into Risk Levels
* Identify potential Dark Pattern impacts
* Generate Business Intelligence reports
* Build interactive Power BI dashboards
* Support business decision-making through analytics

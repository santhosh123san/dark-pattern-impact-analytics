## Project Overview

Dark patterns are deceptive user interface techniques used by websites and mobile applications to influence user decisions.

This project aims to identify, analyze, and measure the impact of dark patterns on user trust, sentiment, retention, and business metrics.

---

## Objectives

- Collect and analyze app reviews
- Detect dark patterns across consumer applications
- Perform sentiment analysis on user reviews
- Build a Trust Score Index
- Analyze user retention and churn indicators
- Develop an interactive Power BI dashboard
